## 🚀 Getting Started

### Prerequisites
- Node.js 22+
- Twilio Account (optional for demo mode)
- OpenAI API Key (optional for NLP features)

### Local Development

```bash
# Backend
cd app/backend
npm install
npm run dev    # Starts on http://localhost:4000

# Frontend
# Open app/frontend/dashboard.html in browser
# Or use live server: npx serve app/frontend
```

### Environment Variables (`.env`)
```env
# Twilio (optional)
TWILIO_ACCOUNT_SID=
TWILIO_AUTH_TOKEN=
TWILIO_PHONE_NUMBER=
CALLEE_NUMBER=

# OpenAI (optional)
OPENAI_API_KEY=

# Server
DASHBOARD_PORT=4000
NODE_ENV=production
```

## 📱 Dashboard Sections

### KPI Metrics
- **Total Contacts**: Number of unique customers in the system
- **Calls Handled**: Voice interactions completed
- **Resolutions**: Successful customer engagements
- **Follow-ups Sent**: WhatsApp/SMS messages dispatched
- **Success Rate**: Percentage of positive outcomes
- **Avg Response**: Average time to resolution

### Analytics
- **Interaction Outcomes** Pie Chart: Distribution of positive/follow-up/escalation responses
- **Customer Sentiment** Bar Chart: Emotional tone analysis of interactions
- **Activity Feed**: Real-time log of all customer interactions

### Intelligent Engagement
- Start/Stop automated campaigns
- Configure campaign cadence (every 5s-60s)
- View next priority customer based on AI ranking
- Real-time queue status (queued/called/done)

### Communication Tools
- **Place Calls**: Initiate outbound voice calls
- **Send WhatsApp**: Quick message dispatch for follow-ups

### Interaction History
- Complete timeline of all customer touchpoints
- Filter by:
  - Customer phone number / name
  - Interaction type (voice, WhatsApp, status)
  - Outcome (positive response, needs follow-up, escalation)
  - Sentiment (positive, neutral, negative)
- Export data as CSV for reporting

## 🔧 API Endpoints

### GET Endpoints
- `GET /api/events` — Fetch all interaction events
- `GET /api/stats` — Get aggregated metrics
- `GET /api/campaign/status` — Campaign runner status
- `GET /stream` — Server-Sent Events stream

### POST Endpoints
- `POST /api/call` — Initiate outbound call
- `POST /api/wa` — Send WhatsApp message
- `POST /api/campaign/start` — Start automated campaigns
- `POST /api/campaign/stop` — Stop campaigns
- `POST /api/campaign/reset` — Reset customer queue
